# Customer_behavior-Analysis
This project focuses on analyzing customer behavior using a combination of SQL, Python, and Power BI. The dataset is processed and explored to uncover meaningful patterns such as purchasing trends, discount usage, and category-wise performance.

SQL is used for data extraction and querying, Python (Pandas, NumPy, etc.) is used for data cleaning and analysis, and Power BI is used to build interactive dashboards and visualizations.

The goal of this project is to demonstrate end-to-end data analysis — from raw data to actionable insights — and to showcase practical skills in data handling, querying, and visualization.
